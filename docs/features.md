# Features

IceScope focuses on practical Iceberg inspection workflows for developers, data engineers, and platform teams.

## Available

- Local Iceberg warehouse browsing.
- Namespace and table tree in Explorer.
- Paginated table previews.
- Schema and catalog metadata views.
- SQL Lab with DataFusion query execution.
- Query history stored locally.
- SQLite-backed application state and cache foundations.
- Dark-first desktop UI.

## Experimental

- S3 and MinIO storage access.
- Manifest-based file discovery and file pruning foundations.
- Query result caching and metadata cache tuning.
- DuckDB and Athena routing placeholders.

Experimental features may change between preview releases.

## Planned

- REST Catalog, AWS Glue, Hive Metastore, and Nessie integrations.
- GCS and Azure storage backends.
- Signed installers and automatic updates.
- Rich snapshot timeline and table history views.
- Export improvements and plugin-style catalog extensions.
