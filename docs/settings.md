# Settings

Settings configure IceScope’s desktop behavior and are stored locally.

## General

- Theme: System, Light, or Dark.
- Language.
- Restore previous session.
- Auto check for updates.

## Data Explorer

- Default page size.
- Max query rows.
- Show row numbers.
- Virtual scrolling.

## Query Editor

- Font size.
- Word wrap.
- Line numbers.
- Auto-complete.
- Query history.

## Cache

- Enable metadata cache.
- Cache duration.
- Clear cache.

## Performance

- Parallel loading.
- Background workers.

## Logging

- Log level.
- Open logs folder.
- Clear logs.

## About

The About section shows app version, Rust version, Tauri version, license, GitHub link, and update checks.
