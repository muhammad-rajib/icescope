# Download IceScope

IceScope preview builds are published from GitHub Releases.

[Open the latest release](https://github.com/muhammad-rajib/icescope/releases/latest)

## Latest release

Do not bookmark direct installer filenames during the preview period. Use the latest release page so you always get the artifacts generated by the current release workflow.

## macOS

- Apple Silicon DMG
- Intel DMG

Unsigned preview builds may show a Gatekeeper warning. Only open builds downloaded from the official GitHub Releases page.

## Windows

- Installer EXE
- MSI

Unsigned preview builds may trigger Microsoft SmartScreen. Verify that the download came from the official IceScope repository before continuing.

## Linux

- AppImage
- DEB
- RPM

For AppImage builds, you may need to make the file executable:

```bash
chmod +x IceScope*.AppImage
./IceScope*.AppImage
```

## Build from source

If an installer is not available for your platform, build locally:

```bash
npm ci
npm run build
cargo build --workspace
npm run tauri:build
```

See [Installation](/installation) for platform notes.
