# Roadmap

See `docs/roadmap.md` for the detailed roadmap.

## Near Term

- Stabilize the first public preview release.
- Improve local and S3 catalog reliability.
- Harden packaging and release verification.
- Prepare signed installer and updater strategy.

## Medium Term

- Add REST Catalog support.
- Expand AWS Glue and Athena integration.
- Improve SQL Lab query ergonomics.

## Long Term

- Add Hive Metastore, Nessie, GCS, and Azure support.
- Introduce plugin architecture for catalogs and engines.
- Ship production updater support.
